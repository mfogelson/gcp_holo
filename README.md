# GCP-HOLO
GCP-HOLO is a RL based tree search method for generating high-order linkage graphs that satisfy the path synthesis problem

## Quick start 
```
python train.py
```

## Requirements
* stable_baselines3==1.5.0
* torch==1.11.0
* torch_geometric==2.0.4
* gym==0.21.0
* networkx==2.6.2
* cma==3.1.0
